## Usage

```sh
yarn add eslint @muhmdraouf/eslint-config-react --dev
```

In `.eslintrc`:

```json
{ 
  "extends": "@muhmdraouf/eslint-config-react"
} 
```

In `tsconfig.json`:

```json
{
  "extends": "@muhmdraouf/eslint-config-react/tsconfig"
}
```
